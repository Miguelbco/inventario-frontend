Nombre del aprendiz: Luis Miguel Briceño Díaz

Descripción:
Este proyecto corresponde al desarrollo del componente frontend de un sistema de inventario de bebidas utilizando React JS.

Funcionalidades:
- Agregar productos
- Visualizar lista de productos
- Manejo de nombre, precio y cantidad

Tecnologías:
- React JS
- JavaScript
- HTML

Instrucciones de ejecución:
1. npm install
2. npm start

Repositorio GitHub:
